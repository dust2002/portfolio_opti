# 포트폴리오 최적화 분석기 (Portfolio Optimizer)

현대 포트폴리오 이론(MPT) 기반의 Streamlit 웹앱입니다.

## 주요 기능

- **몬테카를로 시뮬레이션**: 수만 번의 랜덤 포트폴리오 생성으로 최적 비중 탐색
- **효율적 투자선 시각화**: 위험-수익 관계를 산점도로 표현
- **최대 샤프지수 포트폴리오**: 위험 대비 수익이 가장 효율적인 포트폴리오
- **최소 위험 포트폴리오**: 변동성이 가장 낮은 안정적인 포트폴리오
- **상관관계 히트맵**: 종목 간 상관관계 시각화
- **CSV 다운로드**: 시뮬레이션 결과 및 최적 포트폴리오 내보내기

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 사용 방법

1. 왼쪽 사이드바에 분석할 종목의 티커 심볼 입력 (예: `AAPL, TSLA, META, NVDA`)
2. 분석 기간 선택
3. 무위험 수익률 설정 (기본값: 3%)
4. 시뮬레이션 횟수 선택 (기본값: 20,000회)
5. **'분석 시작'** 버튼 클릭

## 기술 스택

- **Frontend**: Streamlit
- **Data**: FinanceDataReader (한국/미국 주식 데이터)
- **Analysis**: NumPy, Pandas
- **Visualization**: Matplotlib

## 주의사항

본 분석 결과는 투자 권유가 아닙니다. 투자 판단은 본인의 책임 하에 이루어져야 합니다.
